CodeGuardian is an AI-powered developer tool that analyzes code, detects bugs, generates test cases, and provides optimized refactored code.  
It uses React for the frontend, Node.js/Express for the backend, and OpenAI LLMs to perform code analysis.

## Features (MVP)
- Automated bug detection
- AI-generated unit tests (Jest/PyTest/JUnit)
- Code optimization & refactoring suggestions
- Time & space complexity analysis
- Real-time UI with side-by-side output

## Tech Stack
- **Frontend:** React, Vite, Axios, Tailwind (optional)
- **Backend:** Node.js, Express.js, OpenAI API
- **Other:** Prompt Engineering, REST APIs
